# **App Name**: autopxf

## Core Features:

- AI Post Generation: Generates engaging posts for Facebook and X (Twitter) based on user-provided topics, keywords, or content snippets using generative AI.
- Platform Selection: Allows users to select whether the post is intended for Facebook, X (Twitter), or both, optimizing the content accordingly using a tool.
- Tone Adjustment: Provides options to adjust the tone of the generated post (e.g., formal, casual, humorous) to match the user's brand and audience using generative AI.
- Hashtag Suggestion: Suggests relevant hashtags based on the post's content to increase visibility.
- Post Scheduling: Enables users to schedule posts for specific dates and times.
- Performance Analytics: Provides basic analytics (views, likes, shares) to track post performance.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to represent online communication and dynamism.
- Background color: Dark gray (#28282B) for a sleek, modern feel with good contrast.
- Accent color: Vivid pink (#FF69B4) for highlights and calls to action, conveying creativity and social engagement.
- Body and headline font: 'Inter' for a clean, modern, and readable interface.
- Use modern, minimalist icons related to social media, content creation, and analytics.
- Clean, card-based layout with clear visual hierarchy for easy navigation and content discovery.
- Subtle animations on content loading and interaction to enhance user engagement without being distracting.
